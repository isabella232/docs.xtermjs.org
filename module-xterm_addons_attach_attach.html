Source:

Attaches the given terminal to the given socket.

Parameters:
Name Type Description
term Xterm

The terminal to be attached to the given socket.
socket WebSocket

The socket to attach the current terminal.
bidirectional boolean

Whether the terminal should send data to the socket as well.
buffered boolean

Whether the rendering of incoming data should happen instantly or at a maximum frequency of 1 rendering per 10ms.
