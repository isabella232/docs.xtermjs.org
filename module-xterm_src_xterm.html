refresh(start, end, queue)

Source:
  • src/xterm.js, line 1203

Refreshes (re-renders) terminal content within two rows (inclusive)

Rendering Engine:

In the screen buffer, each character is stored as a an array with a character and a 32-bit integer:

  • First value: a utf-16 character.
  • Second value:
  • Next 9 bits: background color (0-511).
  • Next 9 bits: foreground color (0-511).
  • Next 14 bits: a mask for misc. flags:
    • 1=bold
    • 2=underline
    • 4=blink
    • 8=inverse
    • 16=invisible
Parameters:
Name Type Description
start number

The row to start from (between 0 and terminal's height terminal - 1)
end number

The row to end at (between fromRow and terminal's height terminal - 1)
queue boolean

Whether the refresh should ran right now or be queued
